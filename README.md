# Итоговый проект по теме "Система контроля версий Git"

## Цель

1. Создать программу "Будильник", которая будет:
    - Позволять пользователю установить время будильника
    - Опрашивать пользователя о его предпочтениях для будильника (например, звук)
    - Определять, сработал ли будильник в заданное время
    - При срабатывании будильника выводить сообщение и проигрывать звук или выводить текстовое сообщение
    - Реализовывать индивидуальные предпочтения. Например, будильник по дням недели или специальное сообщение
2. Вам представлен выбор из заданий 2-х уровней сложности. Чем выше уровень сложности, тем более функциональное приложение получится. Вам обязательно нужно выполнить все задания базового уровня сложности и опционально задания повышенного уровня сложности

### Базовый уровень сложности
1. **Ввод данных.**
Пользователь должен ввести время для будильника в формате "ЧЧ:ММ". Также у него есть возможность выбрать звуковое уведомление или текстовое сообщение

2. **Цикл работы программы.**
Программа должна постоянно проверять текущее время и сравнивать его с установленным временем будильника. Если время совпало, будильник срабатывает

3. **Вывод данных**
При срабатывании будильника программа должна вывести сообщение "Время вставать!" или проиграть звук

4. **Дополнительные возможности.**
Возможность установить несколько будильников (для этого использовать список/словарь для хранения времени будильников).Будильник должен работать до тех пор, пока не будет остановлен пользователем


### Повышенный уровень сложности
1. Реализовать возможность задания будильников на несколько дней, включая день недели

1. Добавить возможность выбора звука будильника из предустановленных (например, использование файлов звуков в формате .mp3 или .wav)

1. Сделать возможность настройки повтора будильника (например, через каждые 5 минут)
